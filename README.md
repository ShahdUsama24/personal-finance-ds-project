
# Personal Finance as a Project – Data Science Application

## Project Overview

This project models the average individual as a financial system.
The goal is to build a web application that predicts how a person should manage their finances based on income, risk, country, and financial behavior.

We analyze two use cases:
- Egypt
- United States

---

## Core Idea

Treat a person like a project:

Income → Revenue  
Expenses → Operating Costs  
Debt → Liabilities  
Investments → Assets  
Savings → Liquidity  
Risk → Volatility
